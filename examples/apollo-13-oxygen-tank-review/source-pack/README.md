# Apollo 13 Source Pack

This source pack is a small public-domain ledger for a reproducible Research Lab example. It points to original NASA/NTRS sources and records only the claims needed by the example lanes.

The runtime example does not browse the web. It reads checked local lane fixtures derived from this source pack, writes lane artifacts, synthesizes final artifacts, and validates evidence references.

## Source IDs

| Source ID | Source | Why included |
| --- | --- | --- |
| S1 | NASA, `Report of Apollo 13 Review Board` | Root-cause investigation and corrective-action frame |
| S2 | NASA, `Apollo 13 Mission Report` | Mission timeline, abort frame, and operational recovery summary |
| S3 | NASA/NTRS, `Apollo 13 Technical Air-To-Ground Voice Transcription` | Crew/ground symptom record around the accident |
| S4 | NASA, `Apollo 13 Press Kit` | Planned lunar-landing objectives before the accident |

## Limits

- These are excerpts and source-led notes, not complete source reproductions.
- The example supports a bounded incident-review synthesis only.
- Publication-grade historical claims require reading the original full PDFs.
