# Example Source Context

This public profile does not require any private source folders.

For real projects, source context may come from:

- profile docs
- source excerpts pasted into `00-brief.md`
- public URLs listed in the run brief
- user-provided local files kept out of git
- deliberately sanitized fixtures

Do not commit private source documents, account exports, screenshots, bulky raw inputs, or generated research outputs.

Profile docs are context and hypotheses. They are not external proof. External factual claims still require external citations from public sources, official APIs, app listings, reports, reviews, policies, papers, or other cited evidence.
