# Example Tooling Defaults

This profile uses the generic research tool policy in `docs/TOOL_MENU.md`.

Optional Composio session names for generic research:

```text
research_lab_web
research_lab_synthesis
research_lab_optional_paid_fallbacks
```

Recommended local user identifier for examples:

```bash
export COMPOSIO_USER_ID=research-lab-user
```

These names are examples. Real projects may define their own local session names, but tool access must stay within `docs/TOOL_MENU.md` and the active run brief.
