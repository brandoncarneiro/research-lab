# Incident Review Tooling

Default to local files, official public documents, and normal browser/search inspection.

Allowed when scoped by the run brief:

- local file reads from the run source pack
- official public pages or public records
- command lanes that transform checked local source files into raw lane artifacts

Not allowed by default:

- private account access
- stealth scraping
- paywall bypass
- posting, sending, mutating, or production actions
- model-generated incident facts without source IDs

Any broader tool use must be recorded in the run brief with source, budget, and stop conditions.
